# Mi_Primer_Repositorio
Trabajar con GitHub me permitió comprender la importancia de llevar un control organizado y seguro de mis proyectos. A través de esta plataforma pude experimentar cómo se gestionan los cambios, cómo se documenta el progreso y por qué es fundamental mantener una estructura clara dentro de un repositorio.
También entendí el valor de usar commits descriptivos, crear carpetas bien ordenadas y mantener una buena documentación para que cualquier persona pueda entender el proyecto sin dificultades. En general, esta práctica me ayudó a fortalecer mis habilidades en control de versiones y me mostró por qué GitHub es una herramienta esencial para cualquier desarrollador.
